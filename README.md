# SeeImagePX
基于python和OpenCV的图形界面下图形鼠标处的像素位置和灰度值等信息显示。



## LICENSE

采用MIT License，详见 LICENSE 文件。



## INSTALL

如果想要直接使用功能，请到以下链接下载我们已经编译好的版本：

https://github.com/flowshuo/SeeImagePX/releases



如果想要修改脚本，请确保你的电脑安装python3编程环境：

https://www.python.org/

安装以下依赖库：

opencv-python

numpy



## FAQ

### 如何使用
点击菜单栏 打开 打开图片文件。
下方菜单栏将显示该图片鼠标下像素信息。


### 支持的图片类型
PNG JPG BMP GIF
